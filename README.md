# Matplaneraren

A simple and fast web app for planning weekly meals and getting a clear overview of nutritional values.

## Features

- Search for foods and select ingredients
- Calculate nutrition based on weight
- Add and edit meals
- Plan a full week in one place
- Clean, responsive layout for mobile and desktop

## Built With

- HTML
- CSS
- JavaScript (Vanilla)

## Getting Started

1. Clone the project from GitHub
2. Open the project folder in VS Code
3. Run it with any local server or open the start page directly in your browser
